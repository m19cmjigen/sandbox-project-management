# チケット一覧

チケット一覧ページでは、Jiraのチケット（Issue）を一覧で確認できます。複数の条件で絞り込みやソートが可能で、CSV形式でのエクスポートもできます。

## 画面へのアクセス

ナビゲーションバーの **「チケット」** をクリックします。

## テーブルの列

| 列名 | 内容 |
|---|---|
| ステータス | 遅延ステータスチップ（遅延/注意/正常） |
| キー | JiraのIssueキー（例: PROJ-123）。クリック可能（Jira連携時） |
| サマリ | チケットのタイトル |
| プロジェクト | 所属プロジェクトのキー |
| ステータス | Jiraのワークフローステータス（To Do / In Progress / Done） |
| 期限 | 期日（期日超過の場合は赤文字で太字表示） |
| 担当者 | 担当者名（未設定の場合は「—」） |
| 優先度 | チケットの優先度（未設定の場合は「—」） |
| （リンクアイコン） | Jira連携時に表示。クリックでJiraのチケットページを新規タブで開く |

## 絞り込み

### 遅延ステータスでフィルタ

画面上部のトグルボタンで絞り込みます。

| ボタン | 表示されるチケット |
|---|---|
| すべて | 全チケット |
| 遅延 | 遅延 (RED) チケットのみ |
| 注意 | 注意 (YELLOW) チケットのみ |
| 正常 | 正常 (GREEN) チケットのみ |

### 期限未設定のチケット

「期限未設定のみ」チェックボックスにチェックを入れると、`due_date`が設定されていないチケットのみ表示します。

### プロジェクトでフィルタ

「プロジェクト」ドロップダウンから対象プロジェクトを選択します。

### ステータスカテゴリでフィルタ

「ステータス」ドロップダウンで以下を選択できます。

| 選択肢 | 説明 |
|---|---|
| すべて | ステータスを問わず表示 |
| To Do | 未着手のチケット |
| In Progress | 進行中のチケット |
| Done | 完了済みのチケット |

### 担当者でフィルタ

「担当者」ドロップダウンで担当者を選択して絞り込みます。

## 並び順の変更

テーブルの列ヘッダーにある上下矢印アイコンをクリックするとソートができます。

| ソート可能な列 | 説明 |
|---|---|
| キー | JiraのIssueキーでソート |
| 期限 | 期日でソート（デフォルト: 昇順、期日未設定は末尾） |

ヘッダーを1回クリックで昇順、もう1回クリックで降順に切り替わります。

## ページネーション

1ページあたり25件表示されます。画面下部のページネーションでページを切り替えます。ページを切り替えると自動的に画面上部にスクロールします。

## CSV出力

現在の絞り込み条件で最大100件のチケットデータをCSV形式でダウンロードできます。

### 操作手順

1. 絞り込み条件を設定する（任意）
2. 右上の **「CSV出力」** ボタンをクリックする
3. ファイルが自動的にダウンロードされる（ファイル名: `issues_YYYY-MM-DD.csv`）

### CSVの列構成

| 列 | 内容 |
|---|---|
| キー | JiraのIssueキー |
| サマリ | チケットのタイトル |
| プロジェクト | プロジェクトキー |
| ステータス | ワークフローステータス |
| 期限 | 期日（YYYY-MM-DD形式） |
| 担当者 | 担当者名 |
| 遅延ステータス | RED / YELLOW / GREEN |
| 優先度 | 優先度 |
| タイプ | Issueタイプ |

CSVファイルはBOM付きUTF-8形式で出力されるため、Excel等でそのまま開けます。

## Jiraでチケットを開く

各行の右端にあるリンクアイコン（`VITE_JIRA_BASE_URL`の設定が必要）をクリックすると、Jiraの該当チケットページを新しいタブで開きます。

# 組織一覧

組織一覧ページでは、全社の組織階層と各組織のプロジェクト遅延状況をツリー形式で確認できます。

## 画面へのアクセス

ナビゲーションバーの **「組織」** をクリックします。

## 画面構成

```
┌─────────────────────────────────────────────────────────────────┐
│ 組織一覧                                                        │
│                                                                 │
│  ┌── 開発本部 ─────────────────────────────────────────┐        │
│  │  [RED] 10プロジェクト (遅延2 / 注意3 / 正常5)        │        │
│  │                                                     │        │
│  │  ├── 開発部 [YELLOW] 4プロジェクト                   │        │
│  │  │     └── 第一開発課 [GREEN] 2プロジェクト          │        │
│  │  └── インフラ部 [GREEN] 3プロジェクト                │        │
│  └─────────────────────────────────────────────────────┘        │
│                                                                 │
│  ┌── 営業本部 ─────────────────────────────────────────┐        │
│  │  [GREEN] 5プロジェクト                              │        │
│  └─────────────────────────────────────────────────────┘        │
└─────────────────────────────────────────────────────────────────┘
```

## ツリーの見方

### 組織の階層レベル

組織は最大3階層（レベル0〜2）で管理されます。

| レベル | 例 | 表示位置 |
|---|---|---|
| Lv.0 (本部) | 開発本部、営業本部 | 最上位 |
| Lv.1 (部) | 開発部、インフラ部 | 本部の配下 |
| Lv.2 (課) | 第一開発課 | 部の配下 |

### ステータスバッジ

各組織の横にステータスチップが表示されます。

```
[RED]     配下に遅延プロジェクトがある
[YELLOW]  配下に注意プロジェクトがある（遅延なし）
[GREEN]   すべてのプロジェクトが正常
```

### プロジェクト数

各組織のプロジェクト数と遅延内訳が表示されます。

```
例: 10プロジェクト (遅延2 / 注意3 / 正常5)
```

## プロジェクト一覧への移動

組織名をクリックすると、その組織に属するプロジェクトの一覧ページに移動します。

```
操作: 組織名をクリック
  → プロジェクト一覧ページ（対象組織でフィルタ済み）へ移動
```

プロジェクト一覧ページでは、画面上部に「組織IDでフィルタ中」というバナーが表示されます。「絞り込み解除」ボタンで全プロジェクト表示に戻れます。

## 組織管理への移動

組織の追加・編集・削除を行いたい場合は、ナビゲーションの **「管理」** メニューから「組織管理」ページに移動してください。

詳細は [05. 組織管理](./05_organization-management.md) を参照してください。

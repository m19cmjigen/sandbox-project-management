## 基本的な画面構成

```
┌─────────────────────────────────────────────────────┐
│  ナビゲーションバー                                  │
│  ダッシュボード | 組織 | プロジェクト | チケット | 管理 │
├─────────────────────────────────────────────────────┤
│                                                     │
│  メインコンテンツ                                    │
│                                                     │
└─────────────────────────────────────────────────────┘
```

左上のメニューから各機能に移動できます。

## 遅延ステータスの見方

システム全体で3色のステータスを使って遅延状況を表します。

| ステータス | 色 | 意味 |
|---|---|---|
| 遅延 (RED) | 赤 | 期日を過ぎた未完了チケットがある |
| 注意 (YELLOW) | 橙 | 期日まで3日以内、または期日未設定の未完了チケットがある |
| 正常 (GREEN) | 緑 | 遅延・注意チケットなし |

プロジェクトや組織のステータスは、それに属するチケットのステータスを集計して自動的に決まります。

# プロジェクト一覧

プロジェクト一覧ページでは、全社のJiraプロジェクトをカード形式で確認できます。遅延状況でのフィルタリングや並び順の変更が可能です。

## 画面へのアクセス

ナビゲーションバーの **「プロジェクト」** をクリックします。

## 画面構成

```
┌─────────────────────────────────────────────────────────────────┐
│ プロジェクト一覧                                                 │
│                                                                 │
│  [すべて] [遅延あり] [注意] [正常]          並び順: [名前順 ▼]  │
│                                                                 │
│  100件中 1–12件を表示                                           │
│                                                                 │
│  ┌──────────────┐ ┌──────────────┐ ┌──────────────┐           │
│  │ [RED] PROJ-A │ │[GREEN] PROJ-B│ │[YELLOW] PROJ-C│          │
│  │ プロジェクトA │ │ プロジェクトB│ │ プロジェクトC │          │
│  │ 遅延:3 正常:15│ │ 遅延:0 正常:8│ │ 注意:2 正常:6 │          │
│  │ 🎫 チケット  │ │ 🎫 チケット  │ │ 🎫 チケット  │          │
│  └──────────────┘ └──────────────┘ └──────────────┘           │
│                                                                 │
│                     [ 1 ] [2] [3] ... [9]                      │
└─────────────────────────────────────────────────────────────────┘
```

## プロジェクトカードの見方

各カードには以下の情報が表示されます。

| 要素 | 説明 |
|---|---|
| 上部カラーバー | 遅延ステータスに応じた色（赤/橙/緑） |
| プロジェクトキー | Jiraのプロジェクトキー（例: PROJ-A） |
| プロジェクト名 | プロジェクトの名称 |
| ステータスチップ | 遅延/注意/正常 |
| チケット数 | 遅延/注意/正常の件数と合計 |
| チケット一覧ボタン | アイコンボタン。クリックでチケット一覧へ移動 |

## フィルタリング

画面上部のトグルボタンで遅延ステータスで絞り込みができます。

| ボタン | 表示内容 |
|---|---|
| すべて | 全プロジェクトを表示 |
| 遅延あり | 遅延 (RED) プロジェクトのみ |
| 注意 | 注意 (YELLOW) プロジェクトのみ |
| 正常 | 正常 (GREEN) プロジェクトのみ |

## 並び順の変更

右上の「並び順」ドロップダウンで表示順を変更できます。

| 選択肢 | 説明 |
|---|---|
| 名前順 (A→Z) | プロジェクト名のアルファベット昇順 |
| 名前順 (Z→A) | プロジェクト名のアルファベット降順 |
| 遅延数の多い順 | 遅延チケット数が多いプロジェクトを上位表示 |

## ページネーション

1ページあたり12件表示されます。画面下部のページネーションで前後のページに移動できます。

```
[ << ] [ < ] [1] [2] [3] ... [9] [ > ] [ >> ]
```

## チケット一覧への移動

プロジェクトカードのチケットアイコンボタンをクリックすると、そのプロジェクトのチケット一覧ページに移動します。

```
操作: チケットアイコンボタンをクリック
  → チケット一覧ページ（対象プロジェクトでフィルタ済み）へ移動
```

## 組織フィルタからの遷移

組織一覧ページや組織管理ページから特定の組織を指定して遷移してきた場合、画面上部に「組織IDでフィルタ中」というバナーが表示されます。

- **「絞り込み解除」** ボタン: 全プロジェクト表示に戻る
- **「組織一覧へ戻る」** ボタン: 組織一覧ページに戻る

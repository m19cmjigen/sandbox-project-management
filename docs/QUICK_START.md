# クイックスタートガイド

このガイドでは、全社プロジェクト進捗可視化プラットフォームを初めて使う方向けに、基本的な使い方を5分で説明します。

## 1. ログイン（1分）

1. ブラウザでシステムのURLにアクセス
2. **ユーザー名**と**パスワード**を入力
3. **ログイン**ボタンをクリック

```
デフォルト管理者アカウント（初回セットアップ時のみ）:
ユーザー名: admin
パスワード: admin123

⚠️ 本番環境では必ずパスワードを変更してください！
```

## 2. ダッシュボードを確認（1分）

ログイン後、最初に表示されるのがダッシュボードです。

### 見るべきポイント

- 📊 **統計カード**: プロジェクト数、チケット数、遅延状況
- 🎨 **ヒートマップ**: プロジェクトを色分けで表示
  - 🟢 緑 = 順調
  - 🟡 黄 = 注意
  - 🔴 赤 = 遅延

### やってみよう

- ヒートマップのプロジェクトをクリック → 詳細表示

## 3. プロジェクトを見る（1分）

左メニューの**プロジェクト**をクリック

### できること

- 🔍 **検索**: プロジェクト名やキーで検索
- 🎯 **フィルタ**: 組織やステータスで絞り込み
- 📋 **詳細**: カードをクリックして詳細表示

### 試してみよう

1. 検索バーにプロジェクト名を入力
2. 組織フィルターで特定部署のプロジェクトを表示
3. ステータスフィルターで「RED」を選択 → 遅延プロジェクトだけ表示

## 4. チケットを確認（1分）

左メニューの**チケット**をクリック

### 画面の見方

| 列 | 内容 |
|----|------|
| キー | PROJ-123のような識別子 |
| サマリー | チケットの内容 |
| ステータス | ToDo、In Progress、Done等 |
| 遅延 | 🟢🟡🔴で表示 |

### フィルタを使おう

1. **組織で絞り込み**: 自分の部署のチケットだけ表示
2. **プロジェクトで絞り込み**: 特定プロジェクトのチケットだけ
3. **遅延ステータスで絞り込み**: 🔴赤いチケットだけ表示

## 5. 組織構造を理解（1分）

左メニューの**組織管理**をクリック

### ツリー表示

```
本社
 ├── 営業部
 │   ├── 東京営業所
 │   └── 大阪営業所
 └── 開発部
     ├── フロントエンドチーム
     └── バックエンドチーム
```

### 操作方法

- **▶をクリック**: 下位組織を展開
- **▼をクリック**: 下位組織を折りたたみ

## 次のステップ

### 閲覧者の方

これで基本操作は完璧です！
- ダッシュボードで全体を把握
- プロジェクトとチケットで詳細を確認
- 定期的にチェックして進捗を追跡

### マネージャー・管理者の方

管理機能も使えます：

1. **管理画面を開く**
   - 左メニュー > 管理

2. **組織を作成**
   - 組織管理タブ > 新規作成

3. **プロジェクトを組織に割り当て**
   - プロジェクト割り当てタブ

4. **Jira同期を実行**
   - Jira同期管理タブ > 同期を実行

## よく使う機能

### パスワード変更

1. 右上のアカウントアイコンをクリック
2. パスワード変更を選択
3. 現在のパスワードと新しいパスワードを入力

### ログアウト

右上のアカウントアイコン > ログアウト

### 検索のコツ

- **部分一致**: "プロジェ"で"プロジェクト名"が見つかる
- **大文字小文字**: 区別されません
- **即座に反映**: 入力するとすぐに結果が絞られる

## トラブル時の対処

### ログインできない
→ ユーザー名とパスワードを再確認

### データが表示されない
→ ページをリロード（F5キー）

### 「権限がありません」と表示される
→ 自分のロールを確認（右上のバッジ）
- 🔵 閲覧者 = 閲覧のみ
- 🟠 マネージャー = 組織・同期管理可能
- 🔴 管理者 = 全機能使用可能

### それでも解決しない
→ システム管理者に連絡

## 便利な機能

### 🔄 自動更新

データは定期的に自動更新されます：
- Jira同期: 1時間ごと
- 画面表示: リアルタイム

### 📱 レスポンシブ対応

タブレットでも使えますが、PCを推奨

### 🔒 セキュリティ

- 全アクションが監査ログに記録
- ロールベースのアクセス制御
- パスワードは暗号化保存

## 優先度
高

## カテゴリ
Backend

## 説明
ダッシュボード表示に必要な集計データを返すAPIを実装する。

## タスク
- [ ] GET /api/dashboard/summary - 全社サマリの取得
  - 組織ごとの遅延プロジェクト率
  - 組織ごとの総プロジェクト数
  - 組織ごとの遅延チケット数
- [ ] GET /api/dashboard/projects/:org_id - 組織配下のプロジェクトカード情報
  - プロジェクトごとの遅延チケット数
  - プロジェクトごとの期限切れ間近チケット数
- [ ] 集計クエリの最適化
- [ ] キャッシュ戦略の検討・実装
- [ ] APIドキュメント作成
- [ ] パフォーマンステスト

## 受け入れ基準
- すべてのエンドポイントが実装されていること
- 集計結果が正確であること
- パフォーマンス要件を満たすこと（1.5秒以内）
- 統合テストが合格していること

# INFRA-001: AWS アカウント・環境セットアップ

## 優先度
高

## カテゴリ
Infrastructure

## 説明
プロジェクトで使用するAWS環境のセットアップを行う。開発・ステージング・本番環境の3環境を構築する。

## タスク
- [ ] AWS アカウントの作成または既存アカウントの確認
- [ ] 環境別のVPC作成（dev/staging/production）
- [ ] IAMロール・ポリシーの基本設定
- [ ] サブネット構成（Public/Private）の設計と作成
- [ ] セキュリティグループの初期設定
- [ ] CloudWatch Logsの有効化

## 受け入れ基準
- 3つの環境（dev/staging/production）のVPCが作成されていること
- 各環境で必要なサブネットが構成されていること
- IAMロールの基本設定が完了していること

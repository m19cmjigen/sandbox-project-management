## 優先度
中

Batch Worker

## 説明
バッチ処理の実行状況を監視し、異常時にアラートを発報する仕組みを構築する。

## タスク
- [ ] CloudWatch Metricsへのメトリクス送信実装
  - 実行時間

- [ ] CloudWatch Alarmsの設定

- [ ] SNSトピックの作成・通知先設定

- [ ] アラート動作テスト

## 受け入れ基準
- メトリクスが適切に送信されていること
- アラームが設定されていること
- 異常時に通知が届くこと
- ダッシュボードで状況が可視化されていること

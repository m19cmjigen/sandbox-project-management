# FRONT-004: プロジェクト一覧表示の実装

## 優先度
高

## カテゴリ
Frontend

## 説明
選択した組織配下のプロジェクトをカード形式で一覧表示する機能を実装する。

## タスク
- [ ] プロジェクト一覧データの取得（API連携）
- [ ] プロジェクトカードコンポーネントの実装
- [ ] カードへの情報表示
  - プロジェクト名
  - 遅延チケット数
  - 期限切れ間近チケット数
  - ステータスインジケーター
- [ ] カードのソート機能（遅延数、名前等）
- [ ] フィルタ機能
- [ ] ページネーションまたは無限スクロール
- [ ] Jiraへのディープリンクボタン
- [ ] レスポンシブグリッドレイアウト

## 受け入れ基準
- プロジェクトがカード形式で表示されること
- ソート・フィルタが正常に動作すること
- Jiraへのリンクが正しく機能すること
- レスポンシブデザインに対応していること

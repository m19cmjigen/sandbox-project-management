# BACK-004: プロジェクト管理APIの実装

## 優先度
高

## カテゴリ
Backend

## 説明
プロジェクトの取得・組織への紐付けを行うREST APIを実装する。

## タスク
- [ ] GET /api/projects - プロジェクト一覧取得（組織フィルタ対応）
- [ ] GET /api/projects/:id - プロジェクト詳細取得
- [ ] GET /api/projects/unassigned - 未分類プロジェクト一覧
- [ ] PUT /api/projects/:id/organization - プロジェクトの組織紐付け
- [ ] プロジェクトリーダー情報からの組織推測ロジック実装
- [ ] ページネーション対応
- [ ] バリデーション処理
- [ ] APIドキュメント作成
- [ ] 統合テスト

## 受け入れ基準
- すべてのエンドポイントが実装されていること
- ページネーションが正常に動作すること
- 統合テストが合格していること

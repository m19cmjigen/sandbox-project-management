# BACK-005: チケット管理APIの実装

## 優先度
高

## カテゴリ
Backend

## 説明
チケット情報の取得とフィルタリングを行うREST APIを実装する。

## タスク
- [ ] GET /api/issues - チケット一覧取得
- [ ] GET /api/issues/:id - チケット詳細取得
- [ ] フィルタリング機能の実装
  - プロジェクトID
  - 遅延ステータス（RED/YELLOW/GREEN）
  - 期限未設定フラグ
  - ステータスカテゴリ
- [ ] ソート機能の実装（期限日、更新日等）
- [ ] ページネーションまたは仮想スクロール対応
- [ ] パフォーマンス最適化（インデックス活用）
- [ ] APIドキュメント作成
- [ ] 統合テスト

## 受け入れ基準
- すべてのエンドポイントが実装されていること
- フィルタ・ソートが正常に動作すること
- 大量データでもパフォーマンス要件を満たすこと（1.5秒以内）
- 統合テストが合格していること
